<b>Transit Accessibility Computations, Cumulative Opportunities Maps, Schedule-Based Arrival/Departure Time</b> <br /><br />
<span style="color: grey;">
•	<b>TO-Cumulative Opportunities (Schedule-Based Arrival):</b> These maps display aggregate measures calculated across all buildings from which a timetable-informed traveler can reach each destination building in the region via public transport. The default measure is the total number of buildings from which a destination can be reached within the maximum travel time. Optional measures allow users to sum selected building attributes, such as calculating the total population that can access each building within the given timeframe.<br />
•	<b>FROM-Cumulative Opportunities (Schedule-Based Departure):</b> These maps present aggregate measures calculated across all buildings reachable by a timetable-informed traveler using public transport, originating from each building in the region at a specific hour. The default measure is the total number of buildings accessible within the maximum travel time. Additional measures can be calculated by summing selected attributes, such as the total number of accessible jobs from each building.<br />
•	<b>ROUNDTRIP-Cumulative Opportunities (Schedule-Based Arrival and Departure):</b> This map presents aggregate measures calculated across all buildings reachable by a public transport return trip for a timetable-informed traveler, starting from each building in the region at a designated hour. The default measure is the total number of buildings accessible within the maximum travel time. Users can select additional measures, such as summing the total number of jobs accessible via a roundtrip from each building.<br />
In all cases, the total trip duration is strictly limited by the specified maximum travel time.
