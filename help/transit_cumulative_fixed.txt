<b>Transit Accessibility Computations, Cumulative Opportunities Maps, Fixed Arrival/Departure Time</b> <br /><br />
<span style="color: grey;">
•	<b>TO-Cumulative Opportunities (Fixed Arrival):</b> These maps present aggregate measures calculated across all buildings from which a traveler can successfully reach each destination building within the region using public transport. By default, this measure represents the total number of buildings from which a destination can be reached within the maximum travel time. Users can optionally select attribute-based measures, such as building’s population and sum up the population that can reach each building within the specified trip time.<br />
•	<b>FROM-Cumulative Opportunities (Fixed Departure):</b> These maps present aggregate measures calculated across all destination buildings that can be reached via public transport from each building in the region, starting at a specific hour. The default measure is the total number of buildings reachable within the maximum travel time. Users can incorporate additional measures, such as building’s population, and sum up the number of jobs accessible from each building within the given timeframe.<br />
•	<b>ROUNDTRIP-Cumulative Opportunities (Fixed Arrival and Departure):</b> These maps present aggregate measures calculated across all buildings from which facilities can be reached via a public transport return trip, originating from each building in the region at a specified hour. The default measure is the total number of buildings reachable within the maximum travel time. Additional measures can be generated, such as the total number of jobs accessible by the return trip from each building.<br />
For all scenarios, the total trip duration cannot exceed the maximum travel time.
